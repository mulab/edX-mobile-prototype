Tutorials & Resources

There are quite a few free online resources available for learning the material covered in this course. Below is a list of some that you may find useful. You can also check the wiki resources page.
  • Ruby:

    • tryruby.org (beginner)

    • rubymonk.com (beginner/intermediate)

    • rubykoans.com (intermediate/advanced)

  • HTML & CSS:

    • W3 Schools (beginner)

  • Git & GitHub:

    • try.github.com (beginner)

    • Understanding Git Conceptually (intermediate)

    • git ready (beginning to advanced articles, e.g. installing Git, pushing and pulling)

    • An Illustrated Guide to Git on Windows

    • Git for Computer Scientists (advanced)

    • Git Magic (advanced)
